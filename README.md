## 📱 Overview

**AndroRAT12** is an advanced Android Remote Administration Tool (RAT) designed for **educational security testing** and **penetration testing**. This tool allows security professionals to test Android device security, understand vulnerabilities, and demonstrate the importance of mobile security.

> ⚠️ **DISCLAIMER**: This tool is for **EDUCATIONAL PURPOSES ONLY**. Use only on devices you own or have explicit permission to test. Unauthorized access is illegal.
